Server-Side Management

Note 1: The design of this tutorial and almost all of its source code comes from The Java™ Tutorials.

If you are interested in coding your networking back-end from scratch, see the java.net and java.nio packages.

Each IDE has a different way of adding a library as a dependency. This page attempts to explain the methods for the most major IDEs.

JGrasp

All Files

1. Select Settings -> PATH/CLASSPATH -> Workspace

2. Click New

3. Input the directory to the NetAPI.jar file in the Directory Path or JAR File space, then click OK

4. Click OK

Projects

1. Right-click your project, then select PATH/CLASSPATH

2. Click New

3. Input the directory to the NetAPI.jar file in the Directory Path or JAR File space, then click OK

4. Click OK

Intellij

1. Right click your project

2. Select Open Module Settings

3. Select dependencies, then click the little + button

4. Select Jars or Dependencies

5. Navigate to the NetAPI.jar file and select OK

6. Select the checkbox to the left of the new module

7. Select OK

Eclipse

1. Right click your project

2. Select Build Path/Select External Archives...

3. Navigate to the NetAPI.jar file and select open
